Title: Assignment 2

This scrapper gets data from >coinmarketcap.com

## Installation

PyPI
```bash
pip install pycoingecko
```

or from source
```bash
git clone https://github.com/man-c/pycoingecko.git
cd pycoingecko
python3 setup.py install
```

## Usage

Go to test folder and launch test.py
```bash
cd test
python3 test.py
```
